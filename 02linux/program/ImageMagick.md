# ImageMagick
  用于显示，转换和编辑光栅图像和矢量图像文件。它支持读写超过200种图像格式，

## 图像格式和特征识别
- [ImageMagick – Command-line Tools: Identify](https://imagemagick.org/script/identify.php)

1. 用法示例

`$ magick identify file.png`
`identify file.png`
获得更加详细信息：
```
magick identify -verbose file.png
或则
identify -verbose file.png
```
