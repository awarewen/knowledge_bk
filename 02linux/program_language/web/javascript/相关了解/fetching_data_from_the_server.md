现代网站和应用程序中的另一个非常常见的任务是从服务器检索单个数据项以更新网页的各个部分，而无需加载整个新页面。这个看似很小的细节对于网站的性能和行为产生了巨大的影响。

- Prerequisites: 先决条件
    JavaScript 基础

- Objecive: 目的
    了解如何从服务器获取数据并使用它更新网页的内容
